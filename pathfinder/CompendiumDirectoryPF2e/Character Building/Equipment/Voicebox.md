# Voicebox

### [[illusion]][[magical]]


**Price** 5 sp; 
**Bulk** 0; **Size** med
**Usage** worn

This box contains small, magical amplifiers that allow a non-verbal character to translate thought into speech, which emits from the necklace. The speech produced matches any language you understand.
