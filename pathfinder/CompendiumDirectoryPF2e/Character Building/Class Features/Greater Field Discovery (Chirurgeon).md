# Greater Field Discovery (Chirurgeon)
## Feat 13
### [[alchemist]]

**Category** classfeature; 




When you use [[Class Features/Quick Alchemy|Quick Alchemy]] to create any type of elixir of life, the creature healed by the item regains the maximum Hit Points possible, instead of rolling to determine the number of Hit Points regained.

*Source: Pathfinder Player Core 2*
