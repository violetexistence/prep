# Abundant Vials
## Feat 17
### [[alchemist]]

**Category** classfeature; 




You're permanently [[Conditions/Quickened|Quickened]], and can use the extra action only to use [[Actions/Quick Alchemy|Quick Alchemy]] to create a quick vial. You can create only one vial with this action, even if you have [[Class Features/Double Brew|Double Brew]] or a similar ability.

*Source: Pathfinder Player Core 2*
