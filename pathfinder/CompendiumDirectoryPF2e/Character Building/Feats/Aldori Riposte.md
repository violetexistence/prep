# Aldori Riposte `pf2:r`
## Feat 6
### [[archetype]]

**Category** class; 



**Prerequisites** Aldori Parry

**Requirements** You are using an [[Equipment/Aldori Dueling Sword|Aldori Dueling Sword]] to parry.
